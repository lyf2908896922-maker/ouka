1 部署指南

1.1 选择部署方式

欧卡2 TMP查询工具支持多种部署方式，您可以根据自己的需求选择最合适的方式：

云服务器部署

将工具部署到自己的云服务器或虚拟主机上，完全控制和定制。

静态网站托管

使用GitHub Pages、Netlify等免费静态网站托管服务，简单快捷。

1.2 云服务器部署步骤

  1. 1

    准备服务器环境

    确保您的服务器已安装Web服务器软件（如Nginx、Apache等），并支持HTML、CSS和JavaScript。

  2. 2

    上传文件

    将以下文件上传到您的服务器网站根目录：

    • index.html（主页面）
    • wechat_guide.html（本指南）
  3. 3

    配置HTTPS

    微信要求网页必须使用HTTPS协议。请确保您的网站已配置SSL证书，启用HTTPS访问。

    重要： 没有HTTPS，微信将无法正常使用分享等功能！
  4. 4

    测试访问

    打开浏览器，访问您的网站地址，确认工具可以正常打开和使用。

1.3 GitHub Pages部署步骤

  1. 1

    创建GitHub仓库

    登录GitHub，创建一个新的仓库，命名为tmp-query-tool或其他您喜欢的名称。

  2. 2

    上传文件

    将工具文件上传到GitHub仓库：

    # 克隆仓库（在本地执行）
    git clone https://github.com/您的用户名/tmp-query-tool.git

    # 复制文件到仓库目录
    cp index.html wechat_guide.html tmp-query-tool/

    # 提交并推送更改
    cd tmp-query-tool
    git add .
    git commit -m "添加TMP查询工具文件"
    git push origin main
  3. 3

    启用GitHub Pages

    在GitHub仓库页面，点击"Settings" → "Pages"，在"Source"选项中选择分支（如main），然后点击"Save"。

    几分钟后，您的网站将可以通过 https://您的用户名.github.io/tmp-query-tool/ 访问。

1.4 配置微信JS-SDK（可选）

如果您需要使用完整的微信分享功能，需要配置微信JS-SDK。这一步需要您有一个微信公众号。

  1. 1

    获取AppID和AppSecret

    登录微信公众平台（mp.weixin.qq.com），在"开发" → "基本配置"中获取AppID和AppSecret。

  2. 2

    配置JS接口安全域名

    在"设置" → "公众号设置" → "功能设置"中，设置"JS接口安全域名"为您的网站域名。

  3. 3

    修改配置文件

    编辑index.html文件，找到以下代码并修改：

    const WECHAT_CONFIG = {
      appId: 'wx1234567890123456', // 替换为实际的AppID
      debug: false,
      jsApiList: [
        'onMenuShareAppMessage',
        'onMenuShareTimeline',
        'showOptionMenu',
        'hideMenuItems'
      ]
    };

    将 wx1234567890123456 替换为您的实际AppID。

  4. 4

    实现签名生成

    注意：完整的JS-SDK配置需要在服务器端生成签名。由于本工具是纯前端应用，默认使用了模拟数据。

    如果您需要完整功能，需要在服务器端实现签名生成逻辑，并通过API提供给前端。

2 使用方法

2.1 基本使用步骤

  1. 1

    访问工具

    在微信中点击分享链接，或扫描二维码，打开欧卡2 TMP查询工具。

  2. 2

    输入查询信息

    在查询框中输入TMP用户名、TMP ID或SteamID64。

    提示： SteamID64可以在Steam个人资料页面查看，或使用第三方工具如steamid.io获取。
  3. 3

    选择查询类型

    从下拉菜单中选择查询类型：

    • 账号信息 - 查询用户的基本信息、状态等
    • 封禁记录 - 查询用户的封禁历史
    • VTC信息 - 查询用户所属的VTC团队信息
  4. 4

    点击查询按钮

    点击"开始查询"按钮，等待查询结果。

  5. 5

    查看结果

    查询结果将显示在页面上，包括用户的账号状态、加入时间、封禁记录等信息。

2.2 高级功能使用

收藏功能

在查询结果页面，点击"收藏"按钮，可以将当前查询的用户添加到收藏列表中，方便下次快速查询。

提示： 收藏记录保存在本地，更换设备后需要重新收藏。
历史记录

工具会自动保存最近的查询记录，点击历史记录中的条目可以快速重新查询。

下拉刷新

在页面顶部向下拉动，可以刷新当前查询结果或重新加载页面。

分享功能

点击页面顶部的分享按钮，可以将当前查询结果分享给微信好友或分享到朋友圈。

提示： 分享功能需要在微信环境中使用。

3 微信功能集成

3.1 添加到微信公众号菜单

如果您有自己的微信公众号，可以将TMP查询工具添加到公众号菜单中，方便粉丝使用。

  1. 1

    登录微信公众平台

    访问 mp.weixin.qq.com，使用公众号账号登录。

  2. 2

    进入自定义菜单设置

    在左侧菜单中点击"自定义菜单"。

  3. 3

    创建菜单

    点击"添加菜单"按钮，输入菜单名称（如"查询TMP"）。

  4. 4

    设置菜单内容

    选择"跳转网页"，在"网页链接"中输入您的TMP查询工具网址。

  5. 5

    保存并发布

    点击"保存并发布"按钮，等待菜单生效（通常需要10分钟左右）。

3.2 集成微信群机器人（可选）

您可以将TMP查询工具与微信群机器人集成，实现通过群聊命令查询TMP账号信息。

以下是使用微信机器人平台（如WeTool、酷Q等）的基本集成步骤：

  1. 1

    选择机器人平台

    选择一个适合的微信机器人平台，如WeTool、酷Q等。

  2. 2

    编写查询命令

    配置机器人响应特定命令（如"#查询TMP 用户名"），调用查询工具。

  3. 3

    返回查询结果

    机器人调用查询工具API，获取结果并发送到群聊中。

注意：使用第三方微信机器人可能违反微信使用条款，请谨慎使用。

5 常见问题

Q: 为什么在微信中打开显示"不安全"提示？

A: 这是因为您的网站没有配置HTTPS证书。微信要求所有网页必须使用HTTPS协议。请为您的网站配置SSL证书，启用HTTPS访问。

Q: 分享功能无法使用怎么办？

A: 分享功能需要正确配置微信JS-SDK。请确保您已在微信公众平台配置了正确的JS接口安全域名，并且修改了index.html中的AppID配置。如果您只是个人使用，也可以直接复制链接分享给好友。

Q: 查询结果不准确或无法查询怎么办？

A: 请检查输入的TMP用户名或ID是否正确。如果问题仍然存在，可能是因为TruckersMP API暂时不可用，请稍后再试。目前版本使用的是模拟数据，实际部署时需要连接真实的TruckersMP API。

Q: 收藏和历史记录在不同设备间不同步怎么办？

A: 收藏和历史记录是保存在本地浏览器中的，无法自动同步到其他设备。如果需要在多台设备上使用，可以考虑将常用的查询结果分享给自己，方便在其他设备上快速访问。

Q: 如何自定义工具的外观和功能？

A: 您可以编辑index.html文件，修改其中的样式、颜色、文字等内容。如果您有编程经验，还可以添加新功能或修改现有功能。

Q: 是否支持其他游戏平台或模组？

A: 目前工具仅支持TruckersMP模组。如果您需要支持其他平台或模组，需要修改代码并集成相应的API。
